jTDS-Decrypter
==============

Decrypt jTDS (JDBC 3.0) Passwords

Usage
----

To get a list of all options use:

    python jtds_decrypt.py -h
    
Example:
	python jtds_decrypt.py -p "a2 a5 b3 a5 92 a5 92 a5 d2 a5 53 a5 82 a5 e3 a5"
